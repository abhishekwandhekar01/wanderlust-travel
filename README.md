# 🌍 Wanderlust – Travel Booking Web App  

A **Full-Stack Web Application** inspired by Airbnb that allows users to explore, list, and manage travel stays.  
Built using **Node.js, Express.js, MongoDB, EJS**, with cloud and map integrations.  

🚀 **Live Demo:** [Wanderlust on Render](https://wanderlust-travel-12a6.onrender.com/listings)  

---

## ✨ Features  

- 🔐 **Authentication & Authorization** – Secure login/signup system  
- 📝 **CRUD Operations** – Create, Read, Update, Delete listings  
- 📷 **Image Uploads & Storage** – Cloudinary integration  
- ⭐ **Review & Rating System** – Users can leave feedback  
- 🗺 **Interactive Maps** – Mapbox for geolocation & property mapping  
- 🎨 **Responsive UI** – EJS templating with layouts and partials  
- ⚡ **Session Management & Flash Messages** – Smooth user experience  
- 🛡 **Error Handling & Middleware** – Stable and secure backend  
- 🏗 **MVC Architecture** – Clean, scalable code structure  
- 🚀 **Deployed on Render** – Easy access from anywhere

## 🚀 Getting Started  

### Prerequisites  
Make sure you have installed:  
- [Node.js](https://nodejs.org/)  
- [MongoDB](https://www.mongodb.com/)  

### Installation  

```bash
# Clone the repo
git clone https://github.com/abhishekwandhekar01/wanderlust-travel.git

# Navigate into the project folder
cd wanderlust-travel

# Install dependencies
npm install



CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_KEY=your_api_key
CLOUDINARY_SECRET=your_api_secret
MAPBOX_TOKEN=your_mapbox_token
MONGODB_URI=your_mongodb_connection_string
SECRET=session_secret


npm start

```
Visit: http://localhost:3000
